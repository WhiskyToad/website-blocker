# WXT + React

This template helps you develop with React in WXT.

## Prerequisites

- Node.js (v16 or higher)
- npm or pnpm

## Getting Started

1. Clone the repository:

```bash
git clone [repository-url]
cd page-blocker
```

2. Install dependencies:

```bash
npm install
# or
pnpm install
```

3. Start development server:

```bash
npm run dev
# or
pnpm dev
```

4. Build for production:

```bash
npm run build
# or
pnpm build
```
